🎯 Key Dependencies Explained
  • Dio (v5.9.1): إرسال طلبات الـ HTTP والتعامل مع الـ Interceptors وإدارة اتصالات الـ REST APIs.
  • Bloc & Flutter Bloc (v9.x): فصل منطق الأعمال (Business Logic) وإدارة الحالة بنمط Cubits بشكل متكامل ومنظم.
  • Skeletonizer (v2.1.3): إظهار مؤشرات تحميل هيكلية (Shimmer) تلقائية وجذابة متلائمة مع حجم العناصر أثناء جلب البيانات.
  • Local Storage (SharedPreferences & Secure Storage): التخزين المحلي الآمن لبيانات الدخول المشفرة والتوكنز والإعدادات الأساسية.
  • easy_localization (v3.0.8): دعم اللغتين العربية والإنجليزية بشكل ديناميكي مع تغيير اتجاه التطبيق بالكامل.
  • Pinput (v6.0.2): حقل إدخال كود التحقق (OTP) مخصص وذو استجابة سلسة مدعوم بالتحقق التلقائي.
